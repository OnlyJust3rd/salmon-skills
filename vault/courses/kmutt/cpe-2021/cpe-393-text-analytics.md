# CPE 393 - Text Analytics

This course provides an introduction to the techniques and methodologies for processing textual data, extracting valuable insights from unstructured text, and conducting effective textual data analysis. Students will gain foundational knowledge and practical skills essential for working with text-based data in various applications.

## Course Competencies

- Apply Regular Expressions (RegEx) to find or replace substrings, filter relevant text, and clean messy text data for NLP models.
- Present text data in graphical formats using libraries like Matplotlib, Seaborn, and NetworkX to search for patterns, trends, and relationships, including word clouds, heatmaps, bar charts, bubble charts, and network diagrams.
- Extract data from web pages by sending HTTP requests, parsing HTML structures, and storing cleaned data using tools like BeautifulSoup, Scrapy, and Selenium.
- Perform textual data cleaning, including noise removal, handling HTML tags, character normalization, and data masking for privacy.
- Conduct linguistic preprocessing of text, including tokenization, Part-of-Speech (POS) tagging, lemmatization, and Named-Entity Recognition.
- Convert natural language into numerical vectors using discrete representations like One-Hot encoding and Bag-of-Words/TF-IDF, and distributed/continuous representations like Co-Occurrence matrices, Word2Vec (CBOW and Skip-Gram), and GloVe.
- Implement text classification techniques using algorithms such as Naïve Bayes, CNNs, RNNs, LSTMs, and BERT, and evaluate their performance using metrics like Accuracy, Precision, Recall, and F1-score.
- Generate concise text summaries using extractive methods like LexRank and abstractive methods employing Neural Encoder-Decoder frameworks and Transformers, and evaluate summaries using ROUGE scores.
- Group textual data based on similarity using unsupervised clustering methods, including calculating distance functions and applying the K-means clustering algorithm.
- Discover latent themes within a collection of documents using unsupervised topic modeling techniques such as Latent Semantic Analysis (LSA) and Latent Dirichlet Allocation (LDA).
