# CPE 232 - Data Models

CPE232 Data Models (also known as Data Modeling and Visualization) provides a comprehensive introduction to the data science process and how to systematically extract useful knowledge from data to solve problems. You will learn to use Python and essential libraries like Numpy, pandas, and matplotlib to analyze data. Throughout the course, you will concretely learn how to: Identify and work with different types of data, including structured (tabular), semi-structured (JSON, HTML), and unstructured data (text, images, voice). Perform data cleansing and preparation to fix errors, format data, and handle missing values to ensure data quality. Conduct Exploratory Data Analysis (EDA) to detect abnormalities, check assumptions, and discover relationships among variables. Simulate data distributions, such as using Monte Carlo simulations. Apply core Machine Learning (ML) techniques, including linear regression, classification, and clustering. Apply the Cross-Industry Standard Process for Data Mining (CRISP-DM) framework to real-world scenarios. The course culminates in a term project where you will work in a team of five to acquire a real-world dataset, prepare the data, create visualizations, build and evaluate a predictive model, and present your findings through a technical report and video presentation.

## Course Competencies

- Evaluate and apply suitable data modeling techniques to analyze real-world data.
- Create meaningful visualizations using Python and essential libraries like matplotlib that address relevant problems.
- Understand the data science process and the role of data scientists.
