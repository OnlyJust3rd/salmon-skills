# CPE 494 - System Virtualization and Platform Engineering

System virtualization and platform engineering concepts for building, deploying, and managing virtualized and cloud-based systems.

## Course Competencies

- Explain foundational Linux system administration concepts.
- Utilize fundamental Linux commands for file and directory management.
- Apply system thinking to maintain system availability, predictability, and recoverability.
- Monitor system performance and manage jobs and processes.
- Schedule cron jobs for automated tasks.
