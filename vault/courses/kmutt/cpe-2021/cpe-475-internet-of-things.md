# CPE 475 - Internet of Things

This course covers the fundamentals, technologies, and standards of the Internet of Things (IoT), including application domains and architectural components. Topics include smart cyber-physical devices and sensor networks, communication protocols, cloud computing layers, application logic, data analytics, and system implementation.

## Course Competencies

- Explain the foundations, state-of-the-art research, technologies, and standards for the Internet of Things (IoT).
- Describe the application domains and architectural components of IoT.
- Identify and discuss business aspects of IoT.
- Analyze the principles of sensors, actuators, and signal transduction in IoT devices.
- Design and implement embedded systems for IoT nodes using microcontrollers, SBCs, and GPIO programming.
- Compare and contrast various wired and wireless communication technologies used in IoT, including UART/SPI/I²C, Wi-Fi, BLE, and ZigBee.
- Evaluate the characteristics and use cases of IoT application layer data transport protocols such as MQTT, CoAP, and HTTP/REST.
- Explain the architecture of IoT cloud platforms, including device provisioning and API keys, using Azure IoT Hub as a case study.
- Orchestrate data for enterprise applications using REST APIs and JSON data formats, and design effective visual dashboards.
- Apply principles of Edge and Fog computing for distributed intelligence and low-code workflow design in IoT systems.
- Analyze large volumes of IoT telemetry data using techniques for time-series analysis, classification, and clustering.
- Identify and mitigate security and privacy threats in IoT ecosystems, including TLS, authentication, and secure firmware updates.
- Discuss emerging future trends in IoT, such as Digital Twins, 5G IoT, Edge AI/TinyML, Industry 4.0, and sustainability concerns.
