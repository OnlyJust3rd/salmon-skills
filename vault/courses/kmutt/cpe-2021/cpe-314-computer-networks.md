# CPE 314 - Computer Networks

This course introduces fundamental concepts, protocols, and technologies in TCP/IP networks. Topics covered include network architectures, the TCP/IP protocol suite, socket programming, application layer protocols, TCP, UDP, network performance metrics, network layer protocols, routing algorithms and protocols, data link control, multiple access, Ethernet, virtual LAN, wireless LAN, physical transmission media and properties, and network management.

## Course Competencies

- Explain fundamental concepts of computer network architectures and protocol stacks.
- Describe reliable data delivery mechanisms in computer networks.
- Implement socket programming for application layer protocols.
- Analyze application layer protocols such as HTTP and DNS.
- Differentiate between TCP and UDP protocols and their use cases.
- Evaluate network performance metrics and congestion control mechanisms.
- Explain IP addressing, forwarding, subnetting, and ICMP.
- Describe static routing and forwarding principles.
- Explain the functionality of DHCP, NAT, IPv6, VPN, and VPC.
- Compare and contrast intradomain and interdomain routing algorithms and protocols.
- Describe network management concepts and their application in SOHO networks.
- Explain framing, error detection and correction, and multiple access protocols in the link layer.
- Describe Ethernet LANs, LAN interconnection, and VLANs.
- Explain the Spanning Tree Protocol.
- Describe Wireless LANs and PANs.
- Explain multiplexing techniques in the physical layer.
- Describe signals, transmission impairment, and guided and unguided media.
