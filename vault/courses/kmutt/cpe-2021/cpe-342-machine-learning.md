# CPE 342 - Machine Learning

This course explores the theoretical basis of machine learning through examples and hands-on coding exercises. You will study techniques across supervised learning, unsupervised learning, and reinforcement learning. The core topics covered include statistical learning, model training, kernel machines, regression, generalized linear models, tree and ensemble models, dimensionality reduction, clustering, neural networks, deep learning (including vision and representation learning applications), and anomaly detection.

## Course Competencies

- Explain the theoretical basis of machine learning, including statistical learning concepts, Bayes classifier, LDA, and QDA.
- Implement model training using direct (OLS) and iterative (gradient descent) approaches through hands-on coding exercises.
- Apply Support Vector Machines (SVMs), including both linear and nonlinear variations, through hands-on coding exercises.
- Analyze regression techniques, including constraints and generalized linear models, through hands-on coding exercises.
- Construct tree-based models, specifically decision trees and regression trees, through hands-on coding exercises.
- Develop ensemble models through hands-on coding exercises.
- Design and implement neural networks and deep learning models through hands-on coding exercises.
- Build convolutional neural networks through hands-on coding exercises.
- Perform dimensionality reduction techniques through hands-on coding exercises.
- Execute clustering algorithms, including K-means, Gaussian mixture models, hierarchical clustering, and density-based clustering, through hands-on coding exercises.
- Identify and implement anomaly detection methods through hands-on coding exercises.
- Apply reinforcement learning techniques through hands-on coding exercises.
