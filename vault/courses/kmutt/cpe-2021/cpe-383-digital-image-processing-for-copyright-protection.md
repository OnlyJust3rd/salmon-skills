# CPE 383 - Digital Image Processing for Copyright Protection

This course presents the concepts and tools used to provide copyright protection mechanisms, with a particular focus on digital images. The curriculum covers four core areas: digital image processing, cryptography, biometrics-based access control, and digital watermarking. You will learn how to design and implement these concepts for practical copyright protection systems. The course also covers copyright protection mechanisms based on Digital Rights Management (DRM). The ultimate learning objective is for students to successfully add a watermark signal into a digital image using the image processing and cryptographic techniques taught in class.

## Course Competencies

- Explain the fundamental problem of copyright in digital media.
- Describe the basics of digital images and their computational representation as grids of pixels.
- Utilize MATLAB to import, manipulate, visualize, and display digital images.
- Apply point processing operations to modify individual pixels independently.
- Perform spatial transformations on images, including scaling, translation, and rotation.
- Implement neighborhood processing techniques for effects like blurring, sharpening, and edge detection.
- Analyze images in the frequency domain using the Fourier Transform.
- Understand and process different color models (e.g., RGB, HSV).
- Apply techniques for image coding and compression to reduce file size.
- Differentiate between symmetrical and asymmetrical encryption algorithms.
- Explain the principles of hash functions, MAC, and hybrid encryption.
- Discuss real-world issues surrounding copyright violation.
- Describe the fundamentals of access control.
- Explain MPEG-based compression concepts.
- Explain the principles of biometrics and biometric access control systems.
- Describe the core concepts of digital watermarking and watermarking evaluation methods.
- Implement image watermarking techniques.
- Analyze common attacks on digital watermarks.
- Extend watermarking principles to video sequences.
- Present practical assignments related to copyright protection systems.
- Successfully add a watermark signal into a digital image using image processing and cryptographic techniques taught in class.
