# CPE 372 - High Performance Computing and Cloud Technologies

CPE 372: High Performance Computing and Cloud Technology describes the fundamental methods used to assemble parallel programs. The course covers important program design concepts such as partitioning, mapping, and granularity, and applies these concepts to both shared memory and distributed memory systems. Students will learn parallel programming techniques, conduct performance analysis, and learn how to set up a high-performance computer system. Additionally, the course teaches students how to develop programs for many-core processors and introduces frontier computing systems, such as quantum computing.

## Course Competencies

- Explain the history and architecture of High-Performance Computing (HPC).
- Create a basic parallel application using MPI and utilize point-to-point communication functions in an HPC system.
- Utilize collective communication functions and pack data before sending it in MPI.
- Design parallel applications using Foster's design methodology.
- Analyze the necessity of performance analysis and apply Amdahl's law and Gustafson's law.
- Explore parallel programming models and create a parallel program using OpenMP.
- Develop parallel applications utilizing the Python multiprocessing module.
- Describe the architecture and programming of GPGPUs, Xeon Phi, and AI Processors.
- Introduce cluster middleware on GRID Computing, Slurm, and Monitoring Systems, alongside Edge Computing Technology.
- Explain 'General Purpose Brain-Inspired Computing Systems' and program with Spiking Neural Networks.
- Provide an overview of Neuromorphic Architecture.
- Explain the basic ideas of serverless architecture and its real-life applications.
