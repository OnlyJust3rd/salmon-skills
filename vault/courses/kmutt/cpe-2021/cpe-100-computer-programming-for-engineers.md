# CPE 100 - Computer Programming for Engineers

Fundamental concepts of programming including data types, conditional execution, iteration, functions, and I/O with programming exercises. Software development as a problem-solving activity. Techniques for producing correct and robust programs including top-down decomposition, hand simulation and hypothesis-based debugging. Weekly laboratory sessions focus on program design and implementation to solve interesting case problems.

## Course Competencies

- Solve real-world problems using a computational approach.
- Communicate algorithms using visual tools such as flowcharts and pseudocode.
- Implement solutions using the C programming language, designing modular and reusable code.
