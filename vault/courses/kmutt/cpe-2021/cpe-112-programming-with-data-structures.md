# CPE 112 - Programming with Data Structures

Computer programming course with an emphasis on analyzing algorithm efficiency (Big O notation) and implementing dynamic data structures such as arrays, linked lists, stacks, queues, trees, graphs, and hash tables. Creation of general, reusable modules and their use in multi-module software systems using Java packages. Concepts of procedural and data abstraction, encapsulation, information hiding, and object-oriented programming (OOP) using the Java programming language. Weekly lab sessions.

## Course Competencies

- Explain the properties and applications of various data structures, including memory management (static, dynamic, stack, heap, pointers), structural classification (linear vs. non-linear), algorithm efficiency using Big O notation, and real-world use cases for structures like stacks, queues, linked lists, and graphs.
- Select and implement appropriate data structures and associated algorithms in Java to efficiently solve programming problems, including linear data structures (arrays, linked lists, stacks, queues), hierarchical data structures (Binary Trees, BST, AVL Trees with rotations), and graph representations (Adjacency Matrices, Adjacency Lists) with algorithms like DFS, BFS, and Topological Sorting, and perform core operations (traversing, inserting, searching, merging, deleting).
- Utilize object-oriented programming principles in Java to effectively implement data structures, including defining and using Abstract Data Types (ADTs), creating and using Java classes, objects, constructors, and access modifiers, and applying encapsulation for information hiding through private modifiers and controlled getter/setter methods.
