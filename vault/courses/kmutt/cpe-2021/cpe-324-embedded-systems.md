# CPE 324 - Embedded Systems

This course explores the fundamentals and applications of embedded systems through hands-on hardware and software design. Topics include AVR microcontroller architecture, bare-metal C programming, and the configuration of internal peripherals such as I/O ports, Timers/Counters, Analog-to-Digital Converters (ADC), and hardware Interrupts. Students will learn to interface with various sensors and actuators using standard communication protocols (UART, SPI, and I2C). The course also introduces real-time operating systems (RTOS), low-power computing, and reliable system design. Experiments focus on practical engineering skills, progressing from circuit simulation (using Tinkercad) to physical breadboarding, soldering, and custom Printed Circuit Board (PCB) design, emphasizing hardware-level design rather than proprietary approaches.

## Course Competencies

- Explain the fundamental architecture and hardware elements of AVR microcontrollers, including CPU core, registers, memory, I/O port structures, and electrical characteristics.
- Develop bare-metal embedded software in C to control internal AVR microcontroller peripherals, including I/O ports, interrupts, timers/counters, and the Analog-to-Digital Converter (ADC).
- Interface AVR microcontrollers with external sensors, actuators, and communication buses using standard protocols such as UART, SPI, and I2C.
- Apply practical hardware engineering skills, from circuit simulation using Tinkercad to physical breadboarding, soldering, and Printed Circuit Board (PCB) design.
- Design and implement a complete embedded system, incorporating bare-bone Real-Time Operating Systems (RTOS) and utilizing AVR microcontrollers, to meet specific application requirements.
