# MTH 234 - Linear Algebra

MTH 234 Linear Algebra is an undergraduate-level course designed for students who need to understand and use basic linear algebra in practical applications. The course is divided into three main areas: a review of matrices and systems of linear equations, an introduction to vector spaces, and the fundamental theorems of linear transformations.

## Course Competencies

- Define matrices, perform matrix addition, scalar multiplication, and matrix multiplication.
- Set up systems of linear equations and solve them using Gaussian elimination.
- Explore homogeneous linear systems.
- Find the inverse of a matrix.
- Calculate determinants of matrices.
- Utilize Cramer's Rule to solve systems of linear equations.
- Formally define vector spaces and subspaces.
- Understand linear combinations, spanning sets, and linear independence within vector spaces.
- Learn about basis, coordinate vectors, and calculate the dimension of a vector space.
- Determine how to change the basis of a vector space.
- Determine the row space, column space, and rank of a matrix.
- Explore orthogonality and projections in vector spaces.
- Use the Gram-Schmidt Process to construct orthogonal bases.
- Understand functions from one vector space to another that preserve arithmetic operations.
- Determine the matrix representations of linear transformations.
- Find the kernel, range, nullity, and rank of a linear transformation.
- Apply the Rank-Nullity theorem.
- Calculate eigenvalues and their corresponding eigenvectors.
- Learn how to diagonalize matrices.
- Explore the Jordan Canonical Form.
- Analyze functions to determine if they are positive definite, negative definite, or indefinite.
