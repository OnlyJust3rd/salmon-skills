# CPE 241 - Database Systems

Database concepts focusing on design and implement an application with a database. Relational databases, SQL, Database system analysis, Database design and implementation, Concept of database performance and tuning, and NoSQL databases.

## Course Competencies

- Articulate the concepts of relational databases, SQL, database system analysis, database design and implementation, web application and database (3-tier architecture with web browser interface), Enterprise Resource Planning, and NoSQL databases.
- Design and implement a web application with a database.
