# L1- Define and Identify Data Structures and Concepts

Parent macro: [[skills/programming/data-structures/data-structure-principles/data-structure-principles|Data Structure Principles]]

## Competency Statement

Define and identify fundamental data structures, including static and dynamic memory management concepts, linear and non-linear classifications, stacks, queues, linked lists, graphs, arrays, binary trees, BSTs, AVL trees, hash tables, adjacency matrices, and adjacency lists.
