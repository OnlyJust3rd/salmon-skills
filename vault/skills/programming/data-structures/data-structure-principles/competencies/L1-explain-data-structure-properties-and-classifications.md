# L2- Explain Data Structure Properties and Classifications

Parent macro: [[skills/programming/data-structures/data-structure-principles/data-structure-principles|Data Structure Principles]]

## Competency Statement

Explain the differences between linear and non-linear data structures, the concepts of static and dynamic memory management, and the properties and common use cases for various data structures including arrays, linked lists, stacks, queues, binary trees, BSTs, AVL trees, hash tables, adjacency matrices, and adjacency lists.
