# L2- C Programming Elements Explanation and Application

Parent macro: [[skills/programming/programming-languages/c-programming/c-programming|C Programming]]

## Competency Statement

Explain the syntax, semantics, and application of fundamental C programming constructs, including variables, data types, operators, and control flow statements, to contribute to program logic.
