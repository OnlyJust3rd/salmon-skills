# L3- Implement Communication Code

Parent macro: [[skills/programming/programming-languages/avr-communication-programming/avr-communication-programming|AVR Communication Programming]]

## Competency Statement

Implement bare-metal C code to initiate and manage data communication between an AVR microcontroller and an external device using a specified communication protocol (UART, SPI, or I2C).
