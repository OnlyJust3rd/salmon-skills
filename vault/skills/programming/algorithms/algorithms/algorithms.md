# Algorithms

Understanding, designing, analyzing, and applying algorithms for solving computational problems. Covers core definitions, classifications, and properties of algorithms; algorithm design strategies including divide and conquer, dynamic programming, greedy, backtracking, and branch and bound; and performance analysis using asymptotic notation (time and space complexity). Emphasizes selecting appropriate algorithms for different problem types, mapping real-world problems to algorithmic solutions, and evaluating trade-offs in efficiency and scalability.
