# Algorithm Classification

Differentiating between symmetric key algorithms (classical/modern block ciphers and stream ciphers) and asymmetric key algorithms, and comparing lossless compression algorithms (Huffman coding, Adaptive Huffman, LZW) with lossy compression methods (DPCM, Transform coding).
