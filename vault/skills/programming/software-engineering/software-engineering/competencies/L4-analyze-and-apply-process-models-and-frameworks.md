# L4- Analyze and Apply Process Models and Frameworks

Parent macro: [[skills/programming/software-engineering/software-engineering/software-engineering|Software Engineering]]

## Competency Statement

Analyze the strengths and weaknesses of plan-driven process models (Waterfall, Incremental, Reuse-oriented) and Agile Software Development frameworks (Scrum, Kanban, eXtreme Programming) and apply the most suitable model or framework to a given software project scenario.
