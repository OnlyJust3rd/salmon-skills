Parent macro: [[skills/programming/software-engineering/object-oriented-design/object-oriented-design|Object-Oriented Design]]

## Competency Statement

Explain how object-oriented design principles, including responsibility assignment, modularity, and separation of concerns, contribute to system design, maintainability, and the application of design patterns.
