# L3- Apply OO Design Principles

Parent macro: [[skills/programming/software-engineering/object-oriented-design/object-oriented-design|Object-Oriented Design]]

## Competency Statement

Apply principles of object-oriented design, including responsibility assignment and modular decomposition, to design and implement software modules and systems with clearly defined responsibilities and structures.
