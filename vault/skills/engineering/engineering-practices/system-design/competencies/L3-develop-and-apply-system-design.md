# L3- Develop and Apply System Design

Parent macro: [[skills/engineering/engineering-practices/system-design/system-design|System Design]]

## Competency Statement

Develop a basic system-level design (hardware and/or software) for a defined Computer Engineering problem, applying system-level design principles to create a conceptual architecture.
