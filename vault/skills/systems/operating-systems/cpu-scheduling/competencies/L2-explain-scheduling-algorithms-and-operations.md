# L2- Explain Scheduling Algorithms and Operations

Parent macro: [[skills/systems/operating-systems/cpu-scheduling/cpu-scheduling|CPU Scheduling]]

## Competency Statement

Explain the core principles, operational flow, and decision-making processes of CPU scheduling algorithms: First In, First Out (FIFO), Shortest Job First (SJF), Shortest Time-to-Completion First (STCF), Multi-Level Feedback Queue (MLFQ), and Lottery Scheduling.
