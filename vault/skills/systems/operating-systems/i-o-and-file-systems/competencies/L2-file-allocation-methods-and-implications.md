# L2- File Allocation Methods and Implications

Parent macro: [[skills/systems/operating-systems/i-o-and-file-systems/i-o-and-file-systems|I/O and File Systems]]

## Competency Statement

Describe and differentiate various file allocation methods, including contiguous, linked, and indexed allocation, and explain their implications for data access and storage efficiency.
