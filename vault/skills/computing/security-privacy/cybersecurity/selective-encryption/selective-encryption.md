# Selective Encryption

Classifying and critiquing selective encryption algorithms based on when encryption is performed relative to the compression pipeline: precompression, in-compression, and postcompression. This also includes analyzing the evaluation criteria for selective encryption systems, including tunability, visual degradation, cryptographic security, encryption ratio, format compliance, and error tolerance.
