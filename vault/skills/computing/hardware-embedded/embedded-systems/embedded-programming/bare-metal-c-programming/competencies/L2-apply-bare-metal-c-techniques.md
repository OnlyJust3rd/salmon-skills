# L3- Apply Bare-metal C Techniques

Parent macro: [[bare-metal-c-programming|Bare-metal C Programming]]

## Competency Statement

Write C code using bitwise operations to set, clear, and toggle hardware register bits, and correctly apply the volatile qualifier to variables accessed by interrupts or mapped to hardware.
