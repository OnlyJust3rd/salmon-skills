# L2- Explain Text Representation Principles

Parent macro: [[text-feature-engineering|Text Feature Engineering]]

## Competency Statement

Explain the principles behind discrete and distributed text representation methods, including Bag-of-Words, TF-IDF, Word2Vec, GloVe, and document-term matrix construction, and how these representations enable similarity calculations.
