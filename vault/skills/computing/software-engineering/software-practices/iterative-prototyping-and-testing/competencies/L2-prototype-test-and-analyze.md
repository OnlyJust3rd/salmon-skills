# L3- Prototype, Test, and Analyze

Parent macro: [[skills/computing/software-engineering/software-practices/iterative-prototyping-and-testing/iterative-prototyping-and-testing|Iterative Prototyping and Testing]]

## Competency Statement

Execute basic prototype development, conduct tests to assess performance against specified requirements, and employ fundamental mathematics and physics concepts to analyze performance and guide iterative improvements for a Computer Engineering problem.
