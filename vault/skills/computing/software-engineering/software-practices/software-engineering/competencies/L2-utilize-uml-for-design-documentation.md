# L3- Utilize UML for Design Documentation

Parent macro: [[skills/computing/software-engineering/software-practices/software-engineering/software-engineering|Software Engineering]]

## Competency Statement

Utilize UML (Unified Modeling Language) to document design decisions, including static structural aspects with Class and Component diagrams, and dynamic behavioral aspects with Sequence and Activity diagrams.
