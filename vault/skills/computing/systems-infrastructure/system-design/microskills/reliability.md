# Reliability

Explain whether a system continues performing correctly despite bugs, hardware failures, network issues, or dependency outages.

## Real-World Usage

Used when identifying failure modes and designing retries, redundancy, backups, and recovery paths.

## Skill Connection

This microskill supports [[skills/computing/systems-infrastructure/system-design/system-design|system-design]] by giving learners one concrete concept they can use in architecture discussions, interview prompts, and production design reviews.
