# L2-explain-system-design-choice

Learner can explain basic system design choices and justify why a component, pattern, or technology may be selected for a given system need.

## Hirable Usage

This is hirable at junior to early mid level when a candidate can explain why a design uses a cache, load balancer, database, index, or partition instead of only listing buzzwords.
