# L3-design-scalable-service-architecture

Learner can produce a coherent service architecture for a real product scenario and connect components to explicit requirements and failure modes.

## Hirable Usage

This is hirable at mid to senior backend/full-stack level when a candidate can design a URL shortener, feed, notification service, file upload pipeline, or learning graph API and justify the architecture from requirements.
