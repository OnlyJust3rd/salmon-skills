# L3- Apply and Analyze Scheduling Algorithms

Parent macro: [[skills/computing/systems-infrastructure/operating-systems/cpu-scheduling/cpu-scheduling|CPU Scheduling]]

## Competency Statement

Apply CPU scheduling algorithms including FIFO, SJF, STCF, MLFQ, and Lottery Scheduling to a given set of processes and analyze their execution using performance metrics such as turnaround time and waiting time.
