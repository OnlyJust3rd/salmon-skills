# Single Server Architecture

Describe a basic application deployed on one server with one database, including its simplicity, limits, and failure risks.

## Real-World Usage

Used when sketching an MVP architecture before introducing load balancers, caches, replicas, or queues.

## Skill Connection

This microskill supports [[skills/computing/systems-infrastructure/platform-engineering/system-design/system-design|system-design]] by giving learners one concrete concept they can use in architecture discussions, interview prompts, and production design reviews.
