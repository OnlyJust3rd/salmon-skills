# L1-recognize-system-design-term

Learner can recognize common system design terms and identify their basic meaning in a software architecture context.

## Hirable Usage

This is hirable at intern or junior level when a candidate can name the basic building blocks in a design discussion and avoid confusing concepts such as latency, throughput, availability, and reliability.
