# L3- Apply Text Feature Engineering Techniques

Parent macro: [[skills/data/data-science/text-feature-engineering/text-feature-engineering|Text Feature Engineering]]

## Competency Statement

Apply discrete and distributed text feature engineering techniques, including One-Hot encoding, Bag-of-Words, TF-IDF, Co-Occurrence matrices, Word2Vec, and GloVe, to represent text documents numerically for various machine learning tasks such as classification, clustering, and topic modeling.
