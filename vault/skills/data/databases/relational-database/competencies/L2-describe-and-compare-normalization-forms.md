# L2- Describe and Compare Normalization Forms

Parent macro: [[skills/data/databases/relational-database/relational-database|Relational Database]]

## Competency Statement

Describe and compare different normalization forms (e.g., 1NF, 2NF, 3NF) and their impact on relational database design to reduce redundancy and improve integrity.
