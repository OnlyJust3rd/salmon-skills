# L2- List and Describe Protocol Cryptography Roles

Parent macro: [[skills/security/cybersecurity/applied-cryptography/applied-cryptography|Applied Cryptography]]

## Competency Statement

List and describe the role of applied cryptography within network protocols such as IPSec (AH and ESP in transport/tunnel modes), SSL/TLS, PGP, and Kerberos, including its application at different OSI layers.
