# L5- Evaluate Protocol Effectiveness

Parent macro: [[skills/security/cybersecurity/applied-cryptography/applied-cryptography|Applied Cryptography]]

## Competency Statement

Evaluate the effectiveness and security trade-offs of applied cryptography in network protocols such as IPSec (AH and ESP in transport/tunnel modes), SSL/TLS, PGP, and Kerberos across different OSI layers.
