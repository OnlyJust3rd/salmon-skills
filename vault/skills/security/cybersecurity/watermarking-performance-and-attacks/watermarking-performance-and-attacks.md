# Watermarking Performance and Attacks

Evaluating the inherent trade-offs in watermarking (imperceptibility vs. robustness vs. capacity) and measuring watermarking performance using subjective (MOS) and objective metrics (MSE, PSNR, SSIM). This also includes analyzing potential attacks on digital watermarking systems-including removal, geometric, cryptography, and protocol attacks (e.g., copy attacks)-and explaining the use of benchmark tools like StirMark.
