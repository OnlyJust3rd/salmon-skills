# Number Theory in Cryptography

Applying fundamental number theory concepts-such as modular arithmetic, greatest common divisor (gcd), Euclidean algorithms, and Euler's Phi function-to solve cryptographic problems, potentially represented through pseudocode or mathematical derivations.
