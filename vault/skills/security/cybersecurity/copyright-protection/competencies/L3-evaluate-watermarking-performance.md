# L5- Evaluate Watermarking Performance

Parent macro: [[skills/security/cybersecurity/copyright-protection/copyright-protection|Copyright Protection]]

## Competency Statement

Evaluate the inherent trade-offs in watermarking (imperceptibility vs. robustness vs. capacity) and measure watermarking performance using subjective (MOS) and objective metrics (MSE, PSNR, SSIM).
