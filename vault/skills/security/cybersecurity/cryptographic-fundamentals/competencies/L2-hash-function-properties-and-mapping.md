# L2- Hash Function Properties and Mapping

Parent macro: [[skills/security/cybersecurity/cryptographic-fundamentals/cryptographic-fundamentals|Cryptographic Fundamentals]]

## Competency Statement

Describe the key properties of cryptographic one-way hash functions, such as pre-image resistance, second pre-image resistance, and collision resistance, and explain how they transform variable-length input data into fixed-length output digests.
