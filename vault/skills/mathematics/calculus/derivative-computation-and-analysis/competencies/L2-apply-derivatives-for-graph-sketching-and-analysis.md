# L3- Apply derivatives for graph sketching and analysis

Parent macro: [[derivative-computation-and-analysis|Derivative Computation and Analysis]]

## Competency Statement

Use first and second derivatives to determine increasing/decreasing intervals, concavity, inflection points, and extrema to accurately sketch graphs of curves and analyze function behavior.
