# L2- Analyze and Explain Single-Variable Function Properties

Parent macro: [[function-analysis|Function Analysis]]

## Competency Statement

Analyze the fundamental properties of single-variable functions, including domain, range, continuity, limits, inverse functions, and graphical representations, and explain the conditions that define a differentiable function.
