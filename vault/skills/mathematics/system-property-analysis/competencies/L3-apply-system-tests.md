# L3- Apply System Tests

Parent macro: [[system-property-analysis|System Property Analysis]]

## Competency Statement

Apply mathematical tests to determine if a given system exhibits static/dynamic, causal, stable (bounded input, bounded output), invertible, linear, and time-invariant characteristics.
