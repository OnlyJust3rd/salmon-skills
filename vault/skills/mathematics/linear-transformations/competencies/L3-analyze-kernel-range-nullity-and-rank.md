# L3- Analyze Kernel, Range, Nullity, and Rank

Parent macro: [[linear-transformations|Linear Transformations]]

## Competency Statement

Define, explain the relationship of, and compute the kernel, range, nullity, and rank of a linear transformation, potentially using its matrix representation and applying the Rank-Nullity theorem.
