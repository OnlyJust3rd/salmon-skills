# L3- Perform Vector Calculations

Parent macro: [[vector|Vector]]

## Competency Statement

Calculate vector magnitudes, unit vectors, angles between vectors, and orthogonal projections, and execute calculations for the inner (dot) product, vector (cross) product, and scalar triple product.
