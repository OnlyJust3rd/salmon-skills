# Kubernetes

Deploy and manage containerized applications at scale using Kubernetes. Configure Kubernetes components, including Pods, Deployments, StatefulSets, Jobs, CronJobs, Secrets, Resource Requests/Limits, RBAC, and Namespaces. Utilize Helm charts for packaging and deploying Kubernetes applications. Apply YAML configuration rules for Kubernetes resources.
