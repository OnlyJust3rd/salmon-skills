# Verilog HDL Modeling

Model digital circuits using Verilog Hardware Description Language (HDL) and create a testbench to verify the correctness of the hardware design in software simulation before applying it to a real FPGA. Apply different levels of abstraction in Verilog-specifically gate-level, dataflow, and behavioral modeling-to design combinational and sequential modules.
