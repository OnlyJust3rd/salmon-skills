Parent macro: [[skills/hardware-embedded/electronics-embedded/fpga-implementation/fpga-implementation|FPGA Implementation]]

## Competency Statement

Implement a basic Verilog model on an FPGA and deploy a given combinational logic circuit design on an FPGA development board.
