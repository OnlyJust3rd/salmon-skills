# L2- Explain Arithmetic Algorithms

Parent macro: [[skills/hardware-embedded/electronics-embedded/computer-arithmetic/computer-arithmetic|Computer Arithmetic]]

## Competency Statement

Explain common computer arithmetic algorithms for addition/subtraction, multiplication, and division (e.g., ripple-carry vs carry-lookahead adders, Booth multiplication, carry-save addition, restoring vs non-restoring division).
