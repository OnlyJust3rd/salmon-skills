# L3- Interface Microcontrollers with Peripherals

Parent macro: [[skills/hardware-embedded/electronics-embedded/embedded-system-interfacing/embedded-system-interfacing|Embedded System Interfacing]]

## Competency Statement

Apply UART, SPI, and I2C protocols to interface AVR microcontrollers with provided external sensors and actuators, including specific sensor interfacing via I2C.
