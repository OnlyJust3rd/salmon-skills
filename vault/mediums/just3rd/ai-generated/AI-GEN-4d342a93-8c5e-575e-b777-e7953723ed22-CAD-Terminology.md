# Understanding CAD Terminology

In CAD design, like any specialized field, a common language is essential for clear communication and efficient work. Understanding standard CAD terminology is the first step to mastering CAD modeling principles. This lesson introduces you to some of the most fundamental terms you'll encounter.

## What is CAD Terminology?

CAD terminology refers to the specific words, phrases, and abbreviations used within Computer-Aided Design software. These terms describe commands, features, objects, and concepts related to creating and manipulating digital designs, whether in 2D or 3D.

## Key Terms and Concepts

Let's explore some essential CAD terms:

*   **Sketch:** A 2D profile or drawing used as the basis for creating 3D features. Sketches are constrained to define their geometry precisely.
*   **Feature:** A basic building block of a 3D model. Common features include extrudes, revolves, cuts, fillets, and chamfers.
*   **Extrude:** A command that takes a 2D sketch and extends it into a 3D object by a specified distance.
*   **Revolve:** A command that sweeps a 2D sketch around an axis to create a 3D object.
*   **Constrain:** Rules or relationships applied to sketch geometry (e.g., parallel, perpendicular, coincident, dimension) that define its shape and size precisely.
*   **Origin:** The point (0,0) in a 2D sketch or (0,0,0) in a 3D model, used as a reference for positioning.
*   **Axis:** A line used for rotation in revolve features or as a reference for measurement and placement.
*   **Plane:** A flat surface used for creating sketches or as a reference for placing features.
*   **Profile:** The boundary or outline of a 2D sketch.
*   **Dimension:** Numerical values that define the size and position of geometric elements.

## Practical Example: Designing a Simple Bracket

Imagine you need to design a basic L-shaped bracket.

1.  You would start by creating a **sketch** on a **plane**.
2.  This sketch might consist of two lines forming an "L" shape, which you would **dimension** to specific lengths and angles.
3.  You would then apply **constraints** like "perpendicular" to ensure the lines are at a 90-degree angle and add **dimensions** to define the exact size.
4.  Once your 2D sketch is complete and fully defined by constraints, you would use the **extrude** command to give it thickness, transforming the 2D sketch into a 3D feature.
5.  You might then add another sketch on one of the faces to create holes using a **cut** feature, or round off sharp edges with a **fillet**.
