# CAD Tool Primary Functions

This lesson focuses on understanding the fundamental functions of common tools found in Computer-Aided Design (CAD) software. These tools are the building blocks for creating both two-dimensional (2D) drawings and three-dimensional (3D) models.

### Core Concepts

CAD software provides a digital environment to design and document products. To effectively use these programs, you need to know what each basic tool does. Think of these tools like your pencils, rulers, and erasers in traditional drawing, but with digital precision and capabilities.

Here are some of the most common CAD tools and their primary functions:

*   **Line Tool:** Used to draw straight lines. This is fundamental for creating edges, boundaries, and outlines in both 2D and 3D.
*   **Circle Tool:** Used to draw perfect circles. Essential for designing cylindrical features, holes, or circular profiles.
*   **Rectangle Tool:** Used to draw rectangular shapes. Useful for creating walls, bases, or any rectangular component.
*   **Arc Tool:** Used to draw curved lines. More flexible than the circle tool, allowing for partial circles and various curve shapes.
*   **Trim/Extend Tool:** Used to shorten or lengthen lines and curves to meet at a specific point or another entity. This is crucial for cleaning up designs and creating precise intersections.
*   **Offset Tool:** Creates a parallel copy of a selected line, curve, or shape at a specified distance. Useful for creating wall thicknesses, borders, or duplicate features.
*   **Move/Copy Tool:** Used to reposition or duplicate selected objects within the design space.
*   **Rotate Tool:** Used to turn selected objects around a specified point by a defined angle.
*   **Scale Tool:** Used to resize selected objects, either uniformly or non-uniformly.
*   **Extrude Tool (3D):** Takes a 2D shape and extends it into a 3D solid by a specified distance, creating depth.
*   **Revolve Tool (3D):** Takes a 2D profile and rotates it around an axis to create a 3D solid of revolution, ideal for creating symmetrical objects like vases or shafts.

### Practical Example

Imagine you need to design a simple rectangular table with four cylindrical legs in CAD.

1.  **Tabletop:** You would use the **Rectangle Tool** to draw the outline of the tabletop.
2.  **Legs:** For each leg, you would first use the **Circle Tool** to draw a circle representing the base of the leg. Then, you would use the **Extrude Tool** to give each circle depth, turning them into 3D cylinders.
3.  **Positioning:** You would then use the **Move Tool** to position these cylindrical legs correctly under the tabletop.

If you made a mistake and a line for the tabletop extends beyond the rectangle, you'd use the **Trim Tool** to clean it up. If you wanted to create a border around the tabletop, you'd use the **Offset Tool**.

### Practice Task

Open any basic 2D CAD software (or a 3D CAD software and focus on the 2D sketching environment).

1.  Draw a square using the **Rectangle Tool**.
2.  Inside the square, draw a circle using the **Circle Tool**.
3.  Use the **Trim Tool** to remove the portion of the circle that falls inside the square.
4.  Select the remaining arc of the circle and use the **Move Tool** to reposition it outside the square.
5.  Use the **Offset Tool** to create a line parallel to one of the sides of the square, 10 units away.
