# Curve Sketching Criteria: The Essential Checklist

When we solve applied calculus problems, especially those involving optimization or related rates, visualizing the behavior of functions is crucial. Curve sketching is a powerful tool to understand this behavior, and it relies on a set of key criteria. This lesson focuses on remembering these essential criteria for sketching curves.

### What are Curve Sketching Criteria?

Curve sketching criteria are the specific features of a function that we analyze to accurately draw its graph. By examining these features, we can determine the shape, position, and important points of the curve without needing to plot every single point.

### The Essential Criteria for Curve Sketching

Here's a list of the key criteria you should remember when sketching a curve:

1.  **Domain and Range:**
    *   **Domain:** The set of all possible x-values for which the function is defined.
    *   **Range:** The set of all possible y-values that the function can produce.

2.  **Intercepts:**
    *   **y-intercept:** The point where the graph crosses the y-axis. Found by setting \(x=0\).
    *   **x-intercepts (Roots/Zeros):** The points where the graph crosses the x-axis. Found by setting \(y=f(x)=0\).

3.  **Symmetry:**
    *   **Even Function:** If \(f(-x) = f(x)\), the function is symmetric about the y-axis.
    *   **Odd Function:** If \(f(-x) = -f(x)\), the function is symmetric about the origin.

4.  **Asymptotes:**
    *   **Vertical Asymptotes:** Occur where the function approaches infinity or negative infinity. Typically found at x-values where the denominator of a rational function is zero.
    *   **Horizontal Asymptotes:** Describe the behavior of the function as \(x \to \infty\) or \(x \to -\infty\). Determined by comparing the degrees of the numerator and denominator in rational functions.
    *   **Slant (Oblique) Asymptotes:** Occur in rational functions when the degree of the numerator is exactly one greater than the degree of the denominator.

5.  **Intervals of Increase and Decrease (First Derivative Test):**
    *   Find the first derivative, \(f'(x)\).
    *   Find critical points where \(f'(x) = 0\) or \(f'(x)\) is undefined.
    *   Test the sign of \(f'(x)\) in intervals defined by critical points to determine where the function is increasing (\(f'(x) > 0\)) or decreasing (\(f'(x) < 0\)).

6.  **Local Maximum and Minimum Points (First Derivative Test):**
    *   A local maximum occurs where the function changes from increasing to decreasing.
    *   A local minimum occurs where the function changes from decreasing to increasing.

7.  **Concavity and Inflection Points (Second Derivative Test):**
    *   Find the second derivative, \(f''(x)\).
    *   Find points where \(f''(x) = 0\) or \(f''(x)\) is undefined.
    *   Test the sign of \(f''(x)\) in intervals to determine where the function is concave up (\(f''(x) > 0\)) or concave down (\(f''(x) < 0\)).
    *   An **inflection point** is a point where the concavity changes.

### Practical Scenario

Imagine you're an engineer designing a bridge. You need to understand the shape of a support beam's load-bearing capacity, which can be modeled by a function. Before you can analyze stress points or calculate material needs, you first need to sketch the curve representing this capacity. You would use the criteria listed above to identify where the beam is strongest (local maxima), where its bending changes (inflection points), and how it behaves under extreme loads (asymptotes).
