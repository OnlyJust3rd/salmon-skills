# Boolean Function Mapping to Logic Gates

This lesson explores how fundamental logic gates are combined to represent and implement Boolean functions. Understanding this mapping is crucial for building digital circuits that perform calculations and make decisions.

### What are Boolean Functions?

Boolean functions are mathematical expressions that operate on binary values (0 and 1, often represented as False and True). They are the bedrock of digital logic. A Boolean function takes one or more binary inputs and produces a single binary output.

Think of them like simple rules:

*   **AND:** The output is 1 only if ALL inputs are 1.
*   **OR:** The output is 1 if ANY input is 1.
*   **NOT:** The output is the inverse of the input (0 becomes 1, 1 becomes 0).

### The Building Blocks: Logic Gates

Logic gates are the physical implementations of these Boolean operations. They are electronic circuits that perform a specific Boolean function on one or more input signals. The most basic gates are:

*   **AND Gate:** Outputs 1 if all inputs are 1.
*   **OR Gate:** Outputs 1 if at least one input is 1.
*   **NOT Gate (Inverter):** Outputs the opposite of its single input.
