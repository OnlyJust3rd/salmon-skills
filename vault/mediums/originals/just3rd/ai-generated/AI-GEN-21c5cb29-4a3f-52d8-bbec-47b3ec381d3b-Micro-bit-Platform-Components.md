# Micro:bit Platform Components: Remembering the Basics

This lesson focuses on identifying the fundamental hardware and software elements that make up the Micro:bit platform, a key part of understanding embedded systems.

## What is the Micro:bit?

The Micro:bit is a small, programmable computer designed to make learning about coding and electronics fun and accessible. It's a popular tool for beginners in the world of embedded systems.

## Key Hardware Components of the Micro:bit

Let's break down the essential physical parts you'll find on a Micro:bit board.

### 1. The Processor (The Brain)

*   **What it is:** This is the main chip on the Micro:bit. It's a **microcontroller**, which is a small computer on a single integrated circuit. It executes the code you write and controls all the other components.
*   **On the Micro:bit:** You'll see a larger chip, often with branding like "Nordic" or "nRF". This is the heart of the Micro:bit.

### 2. Input/Output (I/O) Pins

*   **What they are:** These are the connections that allow the Micro:bit to interact with the outside world. You can connect sensors, LEDs, motors, and other electronic components to these pins.
*   **On the Micro:bit:**
    *   **Edge Connector:** The gold-plated edge along the bottom is where you'll find the main I/O pins.
    *   **Numbered Pins (0-9):** These are general-purpose pins that can be programmed for various inputs (like reading a button press) or outputs (like turning on an LED).
    *   **"3V" and "GND" Pins:** These provide power (3 volts) and ground connections to external components.
    *   **"RUN" Pin:** Used for resetting the Micro:bit.

### 3. Onboard Devices

These are components built directly onto the Micro:bit board for easy use.

*   **LED Display:**
    *   **What it is:** A grid of 25 (5x5) red LEDs.
    *   **Purpose:** Used to display messages, images, animations, and other visual feedback.
*   **Buttons (A & B):**
    *   **What they are:** Two programmable push buttons.
    *   **Purpose:** Used as input to trigger actions in your programs (e.g., press button A to move an image, press button B to change a score).
*   **Accelerometer:**
    *   **What it is:** A sensor that detects motion and tilt.
    *   **Purpose:** Allows the Micro:bit to know if it's being shaken, tilted in different directions, or moved.
*   **Compass (Magnetometer):**
    *   **What it is:** A sensor that detects magnetic fields, essentially acting like a compass.
    *   **Purpose:** Helps determine the direction the Micro:bit is facing.
*   **Bluetooth Low Energy (BLE) Radio:**
    *   **What it is:** Built-in wireless communication.
    *   **Purpose:** Enables the Micro:bit to communicate wirelessly with other Micro:bits, smartphones, tablets, and computers.
*   **USB Connector:**
    *   **What it is:** A standard USB port.
    *   **Purpose:** Used for powering the Micro:bit and for programming it (downloading your code).
*   **Battery Connector:**
    *   **What it is:** A dedicated connector for a battery pack.
    *   **Purpose:** Allows the Micro:bit to run without being connected to a computer, making it portable.

## Key Software Elements of the Micro:bit

While hardware is the physical foundation, software makes it all work.

### 1. Firmware

*   **What it is:** This is a special type of software that is pre-installed on the Micro:bit's microcontroller. It's low-level software that controls the basic operations of the hardware.
*   **Purpose:** The firmware allows the Micro:bit to be recognized by your computer when you plug it in via USB and enables it to run the programs you download.

### 2. Programming Environments (IDEs)

*   **What they are:** These are applications or websites where you write, edit, and compile your code before sending it to the Micro:bit.
*   **Common Micro:bit IDEs:**
    *   **MakeCode Editor:** A visual block-based editor that also supports JavaScript and Python. It's very beginner-friendly.
    *   **Python Editor (e.g., Mu editor):** For those who prefer to write code in Python.

### 3. Programming Languages

*   **What they are:** The languages you use to tell the Micro:bit what to do.
*   **On the Micro:bit:**
    *   **Block-based Programming:** Visual blocks that you drag and drop to create your program.
    *   **Python:** A popular, readable text-based programming language.
    *   **JavaScript:** Another text-based programming language.

## Practice Activity

Look at an actual Micro:bit board (or a detailed image of one). Can you point out and name each of the hardware components we've discussed? Try to locate the LED display, buttons, I/O pins, and the USB connector. This visual identification is a key step in remembering these components.
