# SALMON

**Skill-aligned Learning Map ONtology**

SALMON is an open ontology for representing skills, learning outcomes, competencies, prerequisites, and learning paths in a graph-based format.

It is designed to help educators, learners, and developers build interoperable learning maps that connect what people want to learn with the skills and knowledge required to achieve those goals.

---

## Why SALMON?

Modern learners have access to a large amount of learning content, but it is often difficult to understand:

- What skills are required for a goal
- Which concepts should be learned first
- How courses, outcomes, and job requirements connect
- What learning path should be followed
- How learning progress can be represented clearly

SALMON addresses this by providing a shared ontology for modeling learning as a structured map of skills, outcomes, and prerequisite relationships.

---

## Core Idea

SALMON represents learning as a graph.

Instead of treating learning content as isolated resources, SALMON connects learning elements through meaningful relationships such as:

- A skill has related learning outcomes
- A learning outcome requires smaller competencies
- A learning outcome may be a prerequisite for another outcome
- A learning path can be generated by traversing prerequisite relationships
- Learning resources can be attached to skills, outcomes, or competencies
